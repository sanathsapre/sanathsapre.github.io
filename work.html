Core Work

What I build.

End-to-end ownership of camera and video data paths on Linux embedded systems.

📡

Camera & Video Streaming

Built production pipelines using GStreamer, RTSP, and WebRTC. Tuned for low latency and high throughput. Handled encoder configuration, jitter buffers, packet loss recovery, and adaptive bitrate scenarios.

⚙️

System-Level Integration

Worked across kernel, user space, and cloud layers — integrating V4L2 camera drivers with GStreamer pipelines, bridging RTSP/WebRTC to cloud ingest endpoints, and ensuring data flows reliably end-to-end.

🔬

Performance & Debugging

Analysed bottlenecks across CPU, memory, and I/O paths. Used perf, ftrace, and kernel tracing to identify frame drops, DMA stalls, and pipeline latency spikes on resource-constrained embedded targets.

🧩

Driver Development

Developed and debugged Linux kernel modules covering character drivers, IOCTL interfaces, kernel timers, workqueues, and GPIO — see the Driver Lab for documented implementations.

🏗️

BSP & Platform Bring-Up

Worked with Yocto-based BSPs to configure kernel builds, device tree overlays, and bootloader settings for ARM embedded targets. Cross-compiled and deployed modules to BeagleBone Black.

🔗

USB / UVC Cameras

Integrated USB Video Class cameras into Linux pipelines. Handled UVC descriptor parsing, format negotiation, and isochronous USB transfer handling for real-time video capture.
