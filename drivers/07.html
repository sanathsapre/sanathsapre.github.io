BBB GPIO Driver

Real hardware on BeagleBone Black

Overview

The first driver that touches real hardware — the onboard USR0 LED on the BeagleBone Black. Controls GPIO1_21 (global GPIO 53) via the kernel GPIO subsystem.

TargetBeagleBone Black (AM335x)

GPIO Numbering on BBB

The AM335x has 4 GPIO banks (0–3), each with 32 pins. Global GPIO = bank × 32 + pin.

/* USR0 LED: GPIO bank 1, pin 21 */
#define GPIO_53  (53)   /* 1 * 32 + 21 = 53 */
LEDBankPinGlobal GPIO
USR012153

The LEDs are owned by leds-gpio at boot. Release before loading: echo none > /sys/class/leds/beaglebone:green:usr0/trigger

GPIO Lifecycle

/* Init */
gpio_is_valid(GPIO_53);
gpio_request(GPIO_53, "GPIO_53");
gpio_direction_output(GPIO_53, 0);
gpio_export(GPIO_53, false);

/* Exit */
gpio_unexport(GPIO_53);
gpio_free(GPIO_53);

Write Handler

static ssize_t etx_write(struct file *filp,
                         const char __user *buf, size_t len, loff_t *off)
{
    uint8_t rec_buf[10] = {0};
    if (copy_from_user(rec_buf, buf, len) > 0)
        pr_err("copy_from_user failed
");
    if (rec_buf[0] == '1')      gpio_set_value(GPIO_53, 1);
    else if (rec_buf[0] == '0') gpio_set_value(GPIO_53, 0);
    return len;
}

Usage

# Turn LED on
echo 1 > /dev/etx_device

# Turn LED off
echo 0 > /dev/etx_device

# Via sysfs
echo 1 > /sys/class/gpio/gpio53/value
