# Linux Driver Lab

**Embedded Linux Developer | Kernel & Driver Development**

A hands-on exploration of Linux device driver design, progressing from
basic character devices to asynchronous event-driven systems.

---

## What This Project Demonstrates

- Character device driver design
- User ↔ Kernel communication
- Blocking vs non-blocking I/O
- poll/select based event handling
- Workqueues and deferred execution
- Timer-based interrupt simulation

---

## Modules

1. Message Queue Driver  
2. Non-blocking Poll Driver  
3. IOCTL Interface  
4. Kernel Timer Driver  
5. Kernel Workqueue Driver  
6. Workqueue + Poll Integration  
7. Timer Interrupt Driver
