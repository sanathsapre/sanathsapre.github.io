07 — BBB GPIO Driver

Real hardware · gpio_requestAM335xUSR LEDgpio_export

Overview

The first driver that touches real hardware — the onboard USR0 LED on the BeagleBone Black. Moves from simulated events to actual memory-mapped GPIO output. Controls GPIO1_21 (global GPIO 53) via the kernel GPIO subsystem.

GPIO Numbering on BBB

The AM335x has 4 GPIO banks (0–3), each with 32 pins. Global GPIO number = bank × 32 + pin.

/* USR0 LED: GPIO bank 1, pin 21 */
#define GPIO_53  (53)   /* 1 * 32 + 21 = 53 */
LEDBankPinGlobal GPIO

The LEDs are owned by the leds-gpio driver at boot. Before loading this driver, release them: echo none > /sys/class/leds/beaglebone:green:usr0/trigger

GPIO Lifecycle in the Driver

/* Init */
gpio_is_valid(GPIO_53);             /* sanity check */
gpio_request(GPIO_53, "GPIO_53");   /* claim ownership */
gpio_direction_output(GPIO_53, 0);  /* configure as output, start LOW */
gpio_export(GPIO_53, false);        /* expose in /sys/class/gpio/ */

/* Exit */
gpio_unexport(GPIO_53);
gpio_free(GPIO_53);

Write Handler

static ssize_t etx_write(struct file *filp,
                         const char __user *buf, size_t len, loff_t *off)
{
    uint8_t rec_buf[10] = {0};

    if (copy_from_user(rec_buf, buf, len) > 0)
        pr_err("ERROR: copy_from_user failed\n");

    if (rec_buf[0] == '1')
        gpio_set_value(GPIO_53, 1);   /* LED on */
    else if (rec_buf[0] == '0')
        gpio_set_value(GPIO_53, 0);   /* LED off */
    else
        pr_err("Unknown command: use 1 or 0\n");

    return len;
}

Usage

# Turn LED on
echo 1 > /dev/etx_device

# Turn LED off
echo 0 > /dev/etx_device

# Read current GPIO state
cat /dev/etx_device

# Or via sysfs (exported)
echo 1 > /sys/class/gpio/gpio53/value
cat /sys/class/gpio/gpio53/value

Cross-Compilation

This driver is built on the host (Dell Latitude 5320, Ubuntu 22.04) and deployed to the BBB via SCP. The Makefile targets make host for cross-compilation and make target for native builds on the BBB.

CROSS_COMPILE = arm-linux-gnueabihf-
ARCH = arm
KDIR = /home/sanath/bbb-kernel/build
